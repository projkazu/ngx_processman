#ngx_processman
---------------------------------------
##Nginxモジュール。 インストール後もNginxモジュール追加可能になります。.htaccessを読むこともできるようになります。PHPはインストールされていれさえすればインストール後、設定なしで使えるようになります。また、SuExecらしきものが使えるようになります。
